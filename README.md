# PrayerTime-Finder

A responsive web application that provides accurate Islamic prayer times for any location worldwide using the Aladhan API.

## Features

- Search prayer times for any city or country
- Display all five daily prayers (Fajr, Dhuhr, Asr, Maghrib, Isha)
- Real-time data based on current date
- Shows timezone and calculation method
- Fully responsive design for all devices
- Clean and modern interface
- Smooth animations and transitions
- Error handling for invalid inputs

## Installation

```bash
git clone https://github.com/hmahmd/prayertime-finder.git
cd prayertime-finder
```

Open `index.html` in your browser. That's it!

## Usage

1. Enter a city or country name (e.g., Karachi, Dubai, London)
2. Click the "Search" button
3. View prayer times with timezone information
4. See calculation method used for the location

## API Integration

Uses Aladhan API for accurate prayer times:

```
https://api.aladhan.com/v1/timingsByAddress/{date}?address={city}&method=8
```

Method 8: University of Islamic Sciences, Karachi
